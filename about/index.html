About this guide

Foreword

I was inspired by Tony Narlock's The Tao of tmux by take the collection of notes I had written on Linux-based bioinformatic workflows and turn them into a guide for others to use. After all, science is a collaborative effort and what good does documentation do if it's not FAIR (Findable, Accessible, Interoperable, Reusable)?

Document structure

These files are written using GitHub Flavored Markdown (GFM), a superset of the original lightweight markup language with plain text formatting syntax.

Web
